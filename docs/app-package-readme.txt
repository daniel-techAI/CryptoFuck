NOCTURNE downloadable web app

The recommended install is the free hosted version:
https://daniel-techai.github.io/CryptoFuck/

Open that URL in Chrome, Edge, or Safari and choose “Install app” inside NOCTURNE.
The browser-managed installation receives automatic updates and supports the cached
market snapshot when the network is unavailable.

This folder is the same static production build for self-hosting. Serve it from an
HTTPS web server; opening index.html directly from disk does not provide the browser
security context required by service workers or Google sign-in.
